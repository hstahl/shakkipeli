----------SHAKKIPELI----------

Olio-ohjelmoinnin perusteet -kurssin harjoitustyö. Syyllisenä Henri Ståhl, 11-08-2013.

QUICK START
Aja Java virtuaalikoneella tästä kansiosta luokkatiedosto Shakkipeli komennolla "java fi.utu.harjoitukset.shakkipeli.Shakkipeli"
Windows konsolissa kirjoita chcp 1257, jotta saat ääkköset näkyviin.

FEATURES
Komentorivillä pelattava shakkipeli ilman pelin voittamista valvovaa logiikkaa. Tietokonevastustajaa ei ole. Keskeneräisen pelin voi tallentaa tiedostoon ja jatkaa myöhemmin. Bugit ovat featureita.

DOKUMENTAATIO
Löytyy tiedostosta docs\index.html

LÄHDEKOODI
Lojuu kansiossa src

LISÄKSI
Jos jdk:n bin kansio on asetettu ympäristömuuttuja Path:iin, voi käyttää tästä kansiosta löytyviä batch tiedostoja.
run.bat tekee saman kuin QUICK START osiossa ohjeistetaan
compile.bat kääntää lähdetiedostot .class tiedostoiksi oikeisiin kansioihin
buildJavadoc.bat luo javadoc dokumentaatiotiedostot docs -kansioon
